# 🏪 StoreFlow | Entity Framework Core Eğitim Projesi

Bu proje, Entity Framework Core’un tüm gücünü gerçek senaryolarla deneyimlemek amacıyla hazırlanmıştır. LINQ ve EF Core'un 70'ten fazla metodunun pratik uygulamalarını içerir. Proje; müşteri, ürün, kategori ve sipariş gibi varlıklarla çalışır, ilişkisel veritabanı yapıları kurar ve yönetir.

## 🎯 Proje Amacı
Entity Framework Core ile veri yönetimini ileri seviye tekniklerle öğrenmek, LINQ sorgularını profesyonel düzeyde uygulayarak performanslı ve okunabilir kod üretimini sağlamak.

## 🧱 Kullanılan Teknolojiler
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- LINQ
- Code First Yaklaşımı

## 🔍 Öne Çıkan Özellikler
- 🔄 CRUD İşlemleri (Add, Remove, Update, Find)
- 🔗 Navigation Properties & Include Kullanımı
- 📦 Eager, Lazy ve Explicit Loading
- 📊 Dashboard & Grafiksel Raporlama
- 🧠 LINQ Metodlarıyla Gelişmiş Veri Sorgulama
- 📚 Gerçek senaryolarla 70+ LINQ & EF Core metodu

## 🧪 Kapsanan Bazı Metotlar
- `Add`, `AddAsync`, `AddRange`, `AddRangeAsync`
- `Find`, `FindAsync`, `FirstOrDefault`, `SingleOrDefault`
- `Where`, `Include`, `Select`, `SelectMany`
- `Any`, `All`, `Count`, `LongCount`
- `OrderBy`, `OrderByDescending`, `ThenBy`
- `GroupBy`, `Join`, `GroupJoin`
- `Take`, `Skip`, `TakeLast`, `SkipLast`
- `Aggregate`, `Append`, `Prepend`, `Chunk`
- `Union`, `UnionBy`, `Intersect`, `Except`, `ExceptBy`
- `AsNoTracking`, `Attach`, `Entry`

## 📂 Proje Yapısı
- `Entities/`: Varlık sınıfları (Customer, Order, Product...)
- `Data/`: DbContext ve seed veriler
- `Controllers/`: Sayfa kontrolleri
- `Views/`: Razor tabanlı görünümler
- `ViewComponents/`: Parçalı görünümler
